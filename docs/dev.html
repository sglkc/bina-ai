BINA AI

Chrome extension modern dengan dukungan AI untuk meningkatkan produktivitas browsing Anda dengan fitur speech recognition, markdown parsing, dan notifikasi yang elegan.

Instalasi

Untuk menginstall BINA AI Chrome Extension, ikuti langkah-langkah berikut:

Dari Source Code

# Clone repository
git clone https://github.com/username/bina-ai.git
cd bina-ai

# Install dependencies
pnpm install

# Build extension
pnpm run build

# Load ke Chrome
# 1. Buka chrome://extensions/
# 2. Enable "Developer mode"
# 3. Klik "Load unpacked"
# 4. Pilih folder .output/chrome-mv3

Speech Recognition

Fitur speech recognition memungkinkan Anda untuk mengontrol extension menggunakan perintah suara. Komponen ini diimplementasikan dalam entrypoints/popup/components/SpeechRecognition.tsx.

Markdown Parsing

Extension dapat memproses dan menampilkan content markdown secara real-time. Modul ini dapat ditemukan di entrypoints/markdown.content.ts.

Toast Notifications

Sistem notifikasi yang comprehensive dengan audio feedback dan berbagai gaya visual. Implementasi tersedia di entrypoints/toast.content.ts.

Konfigurasi

Extension dapat dikonfigurasi melalui halaman options yang dapat diakses dari popup atau melalui pengaturan Chrome.

API Key

Untuk menggunakan fitur AI, Anda perlu mengatur API key di halaman options. Komponen konfigurasi API key tersedia di entrypoints/options/APIKey.tsx.

# Setup environment variables
cp .env.example .env

# Edit .env file dengan API keys yang diperlukan
OPENAI_API_KEY=your_api_key_here

Cara Penggunaan

Menggunakan Popup

Klik icon extension di toolbar Chrome untuk membuka popup interface. Dari sini Anda dapat:

  • Mengaktifkan speech recognition
  • Mengakses pengaturan extension
  • Melihat status extension

Background Actions

Extension berjalan di background dan dapat melakukan berbagai aksi otomatis seperti:

  • Monitoring aktivitas tab
  • Pemrosesan perintah AI
  • Sinkronisasi data

Pengembangan

Tech Stack

  • Framework: WXT (Web Extension Toolkit)
  • UI: React + TypeScript
  • Styling: UnoCSS
  • Build Tool: Vite
  • Package Manager: pnpm

Development Commands

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Type checking
pnpm run type-check

# Linting
pnpm run lint
